Dynamic Spectrum Allocation Problem

Problem Statement

Given a set of $N_L$ radio links and $N_C$ available channels, the Dynamic Spectrum Allocation Problem requires assigning exactly one channel to each link. Each link-channel pair $(l,c)$ yields a utility score $U_{l,c}$. A subset of link pairs $\mathcal{I} \subseteq \{0,\dots,N_L-1\}^2$ are designated as interfering; if two interfering links $l_1$ and $l_2$ are assigned the same channel $c$, a penalty $P_{l_1,l_2}$ is incurred. The task is to find a channel assignment that maximizes the total utility minus the total interference penalty.

Decision Variables

For each link $l \in \{0, \dots, N_L-1\}$ and channel $c \in \{0, \dots, N_C-1\}$, define a binary decision variable $x_{l,c} \in \{0,1\}$. The variable $x_{l,c}$ equals $1$ if and only if link $l$ is assigned channel $c$, and $0$ otherwise.

QUBO Derivation

Step 1 -- Objective Conversion. The original goal is to maximize utility minus interference. Converting to a minimization problem suitable for QUBO yields: \begin{align} \mathcal{H}_{\text{obj}}(x) &= -\sum_{l=0}^{N_L-1} \sum_{c=0}^{N_C-1} U_{l,c} x_{l,c} + \sum_{(l_1,l_2) \in \mathcal{I}} \sum_{c=0}^{N_C-1} P_{l_1,l_2} x_{l_1,c} x_{l_2,c}. \end{align} Step 2 -- Constraint Formulation. Each link must be assigned exactly one channel. This is enforced by the equality constraints: \begin{align} \sum_{c=0}^{N_C-1} x_{l,c} = 1, \quad \forall l \in \{0, \dots, N_L-1\}. \end{align} Step 3 -- Penalty Expansion. We introduce a penalty weight $\lambda > 0$ and convert each constraint into a quadratic penalty term. For a fixed link $l$, the penalty is: \begin{align} \mathcal{P}_l(x) &= \lambda \left( \sum_{c=0}^{N_C-1} x_{l,c} - 1 \right)^2. \end{align} Expanding the square and applying the idempotent property $x_{l,c}^2 = x_{l,c}$ for binary variables gives: \begin{align} \left( \sum_{c} x_{l,c} - 1 \right)^2 &= \sum_{c=0}^{N_C-1} x_{l,c}^2 + \sum_{c \neq c'} x_{l,c} x_{l,c'} - 2 \sum_{c=0}^{N_C-1} x_{l,c} + 1 \nonumber \\ &= \sum_{c=0}^{N_C-1} x_{l,c} + 2 \sum_{0 \leq c < c' \leq N_C-1} x_{l,c} x_{l,c'} - 2 \sum_{c=0}^{N_C-1} x_{l,c} + 1 \nonumber \\ &= 1 - \sum_{c=0}^{N_C-1} x_{l,c} + 2 \sum_{0 \leq c < c' \leq N_C-1} x_{l,c} x_{l,c'}. \end{align} Multiplying by $\lambda$ yields the general penalty contribution for link $l$: \begin{align} \mathcal{P}_l(x) = \lambda - \lambda \sum_{c=0}^{N_C-1} x_{l,c} + 2\lambda \sum_{0 \leq c < c' \leq N_C-1} x_{l,c} x_{l,c'}. \end{align} Step 4 -- Combined Hamiltonian. The total QUBO Hamiltonian is the sum of the objective and all link penalties: \begin{align} \mathcal{H}(x) = \mathcal{H}_{\text{obj}}(x) + \sum_{l=0}^{N_L-1} \mathcal{P}_l(x). \end{align} Step 5 -- Q Matrix and Offset. Collecting coefficients from $\mathcal{H}(x)$, the QUBO parameters are defined as follows. The constant offset is $c = \lambda N_L$. The diagonal and off-diagonal entries of the upper-triangular Q matrix are: \begin{align} Q_{(l,c),(l,c)} &= -U_{l,c} - \lambda, \quad \forall l \in \{0,\dots,N_L-1\}, c \in \{0,\dots,N_C-1\}, \nonumber \\ Q_{(l_1,c),(l_2,c)} &= P_{l_1,l_2}, \quad \forall (l_1,l_2) \in \mathcal{I}, c \in \{0,\dots,N_C-1\}, \nonumber \\ Q_{(l,c),(l,c')} &= 2\lambda, \quad \forall l \in \{0,\dots,N_L-1\}, 0 \leq c < c' \leq N_C-1, \nonumber \\ Q_{i,j} &= 0, \quad \text{otherwise}. \end{align}

Penalty Weight Justification

The penalty weight $\lambda$ must be chosen sufficiently large to ensure that violating the one-hot constraint is strictly more costly than any potential gain in the objective function. Specifically, we require: \begin{align} \lambda > \max \left( \max_{l,c} |U_{l,c}|, \max_{(l_1,l_2) \in \mathcal{I}} P_{l_1,l_2} \right). \end{align} If a link $l$ violates its constraint, the penalty term $\mathcal{P}_l(x)$ increases by at least $\lambda$. The maximum possible reduction in $\mathcal{H}_{\text{obj}}$ achievable by reassigning channels is bounded by $\max_{l,c} |U_{l,c}|$ (from utilities) or $\max_{(l_1,l_2) \in \mathcal{I}} P_{l_1,l_2}$ (from interference penalties). Since $\lambda$ exceeds both bounds, any infeasible configuration has an energy strictly greater than the best feasible configuration, guaranteeing that the constraint is respected at the optimum.

Correctness Argument

Minimizing $\mathcal{H}(x)$ over $\{0,1\}^{N_L N_C}$ recovers the optimal channel assignment because (i) for any feasible assignment, all penalty terms vanish, reducing $\mathcal{H}(x)$ to $\mathcal{H}_{\text{obj}}(x)$; thus, the minimum over feasible points corresponds exactly to the maximum utility minus interference. (ii) For any infeasible assignment, at least one constraint is violated, adding a penalty of at least $\lambda$ to the energy. By the choice of $\lambda$, this penalty strictly dominates any possible decrease in $\mathcal{H}_{\text{obj}}$ resulting from the infeasibility. Consequently, the global minimum of $\mathcal{H}(x)$ must occur at a feasible point that optimizes the original objective function.
