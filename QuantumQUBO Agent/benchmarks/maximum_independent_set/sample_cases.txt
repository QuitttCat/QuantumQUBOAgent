Test case 4: 5 nodes, complement of the CSPLib/DIMACS clique-format example

Source: Hand curated adaptation from:
  CSPLib Problem 074, Maximum Clique, DIMACS-format example
  Link: https://www.csplib.org/Problems/prob074/
  Challenge citation: David S. Johnson and Michael A. Trick, eds.,
  "Cliques, Coloring, and Satisfiability", DIMACS Series 26, 1996.
  DOI/link: https://doi.org/10.1090/dimacs/026

Number of vertices: 5
Edges:
  (0, 2), (0, 4), (1, 3), (1, 4)

Note: The CSPLib example gives a 5-node graph whose maximum clique is {2, 3, 4}
      in 0-indexed notation after converting from the original 1-indexed DIMACS file.
      Maximum clique in a graph is equivalent to maximum independent set in its complement.

Maximum independent set size = 3

Optimal independent set (unique solution):
  {2, 3, 4}
