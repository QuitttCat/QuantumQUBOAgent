Maximum Independent Set Problem

An independent set in an undirected graph is a subset of vertices in which no two selected
vertices are adjacent.

The Maximum Independent Set Problem asks: given an undirected graph G = (V, E), find the
largest independent set, i.e., the subset S subseteq V of maximum size such that for every
edge (i, j) in E, at most one of i or j is selected.

Objective:
- Maximize the number of selected vertices.
