Brain Connectome Community Detection Problem

Given a weighted brain connectivity graph, assign each brain region to one of a
small number of communities. A good partition has strong within-community
connectivity and weak between-community connectivity.

Task:
Choose one community label for each brain region.

Objective:
Maximize the supplied modularity-like community score.

Goal:
Return the community assignment and maximum score.

QUBO convention:
For two communities, use one binary variable per brain region. For more than two
communities, use one-hot region-community variables or a discrete quadratic
model. Pairwise terms reward assigning strongly connected regions to the same
community and penalize unsuitable cross-community assignments.
