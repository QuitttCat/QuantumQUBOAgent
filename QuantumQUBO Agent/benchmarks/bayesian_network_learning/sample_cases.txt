BAYESIAN NETWORK STRUCTURE LEARNING TEST CASES

Source inspiration:
Yuta Shikuri, "Decomposed Quadratization: Efficient QUBO Formulation for
Learning Bayesian Network", Proceedings of the AAAI Conference on Artificial
Intelligence 39(11):11345-11352, 2025.
DOI: https://doi.org/10.1609/aaai.v39i11.33234

These cases use the score-based Bayesian network structure learning setting.
The score is decomposable: each node contributes the local score of its selected
parent set. The task is to select one and only one candidate parent set for each
node, with exactly K directed edges in total, while maximizing the decomposable
score.

Each node may have a separate number of candidate parent sets. For each node,
the collection listed below is the complete set of admissible parent-set
selections for that node.


========================================================================
TEST CASE 1: 3-node candidate-parent-set instance

Node order:
0 < 1 < 2

Required total directed edges:
K = 2

Candidate parent sets:
Node 0:
- candidate 0: parents {}, score 0

Node 1:
- candidate 0: parents {}, score 0
- candidate 1: parents {0}, score 4

Node 2:
- candidate 0: parents {}, score 0
- candidate 1: parents {0}, score 3
- candidate 2: parents {1}, score 6
- candidate 3: parents {0, 1}, score 7

Ground truth:
- Maximum total score: 10
- QUBO minimization optimal value: -10
- Number of optimal solutions: 1
- Selected parent sets:
  - node 0: parents {}
  - node 1: parents {0}
  - node 2: parents {1}
- Directed edges in the learned Bayesian network:
  - 0 -> 1
  - 1 -> 2
