Job Shop Scheduling Problem

Given jobs made of ordered operations and machines that can process only one
operation at a time, choose start times for all operations.

Task:
Schedule all operations.

Constraints:
- Operations within each job must follow their given order.
- A machine cannot process two operations at the same time.

Objective:
Minimize the makespan, the completion time of the final operation.

Goal:
Return the start time of each operation and the minimum makespan.

QUBO convention:
Use binary variables indicating whether an operation starts at a particular
time. Penalties enforce one start time per operation, precedence constraints,
and no machine conflicts.
