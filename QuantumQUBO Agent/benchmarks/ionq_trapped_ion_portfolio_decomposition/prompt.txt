IonQ Trapped-Ion Portfolio Decomposition Problem

Given assets that have been clustered into hardware-sized subproblems, choose a
fixed number of assets from each subproblem. Each asset has a return score, and
asset pairs have risk/correlation penalties.

Task:
Choose exactly K assets from the cluster.

Objective:
Maximize:

  total portfolio score =
    sum of selected asset return scores
    - sum of selected pairwise risk penalties.

Goal:
Return the selected cluster assets and maximum risk-adjusted score.

QUBO convention:
Use one binary variable per asset in the hardware-sized cluster. A quadratic
penalty enforces the exact cardinality constraint. The cluster-level QUBO is a
subproblem in a larger decomposition/recombination pipeline.
