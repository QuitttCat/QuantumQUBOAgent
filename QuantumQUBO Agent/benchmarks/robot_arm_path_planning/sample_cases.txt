ROBOT ARM PATH PLANNING TEST CASES

Source inspiration:
Shogo Arai and Ryu Ito,
"Path Planning for Robotic Arms using Quantum Annealing",
Proceedings of JSME Annual Conference on Robotics and Mechatronics, 2024.
DOI: https://doi.org/10.1299/jsmermd.2024.1P2-L06

Benchmark simplification:
These cases discretize the arm configuration space into named states and use
explicit transition costs.


========================================================================
TEST CASE 1: Three-step path around obstacle

Time steps: 3
Start state: A at t0
Goal state: D at t2

Allowed states:
- t0: A
- t1: B, C
- t2: D

State obstacle costs:
- B: 3
- C: 0

Transition costs:
- A -> B: 1
- A -> C: 2
- B -> D: 1
- C -> D: 1

Ground truth:
- Minimum path cost: 3
- Number of optimal paths: 1
- Selected path:
  - A -> C -> D
- Score calculation:
  - obstacle cost C: 0
  - motion cost: 2 + 1 = 3


========================================================================
TEST CASE 2: Four-step path with smooth motion

Time steps: 4
Start state: A at t0
Goal state: E at t3

Allowed states:
- t0: A
- t1: B, C
- t2: D, F
- t3: E

State obstacle costs:
- B: 0
- C: 2
- D: 0
- F: 1

Transition costs:
- A -> B: 1
- A -> C: 1
- B -> D: 1
- B -> F: 3
- C -> D: 1
- C -> F: 1
- D -> E: 1
- F -> E: 1

Ground truth:
- Minimum path cost: 3
- Number of optimal paths: 1
- Selected path:
  - A -> B -> D -> E
