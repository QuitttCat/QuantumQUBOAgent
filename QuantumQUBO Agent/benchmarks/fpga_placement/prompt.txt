FPGA Placement Problem

Given logic blocks and discrete FPGA sites, assign each logic block to exactly
one site. Each site can hold at most one block. Connected blocks incur wirelength
or communication costs based on their assigned sites.

Task:
Place all logic blocks on FPGA sites.

Constraints:
- Each block is assigned to exactly one site.
- Each site is used by at most one block.

Objective:
Minimize total weighted communication distance between connected blocks.

Goal:
Return the block-to-site placement and minimum placement cost.

QUBO convention:
Use one binary variable per block-site assignment. One-hot penalties enforce
legal placement, and quadratic terms encode communication costs between assigned
sites for connected block pairs.
