Clinical Enzyme Target Identification Problem

Given a disease-related metabolic network, choose a set of enzyme targets whose
removal or inhibition disrupts disease-associated reactions. Each enzyme has a
target benefit score, and pairs of enzymes can have redundancy penalties if they
affect the same pathway region.

Task:
Choose exactly K enzyme targets.

Objective:
Maximize:

  total target score =
    sum of selected enzyme benefit scores
    - sum of redundancy penalties between selected enzyme pairs.

Goal:
Return the selected enzyme targets and maximum total score.

QUBO convention:
Use one binary variable per enzyme candidate. Cardinality penalties enforce
exactly K selected targets, while quadratic terms encode pathway overlap or
redundancy between enzyme targets.
