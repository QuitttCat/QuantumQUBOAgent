Bin Packing Problem

Given a list of item weights and identical bins with capacity C, assign every
item to exactly one bin while respecting bin capacities.

Task:
Use the fewest bins possible.

Objective:
Minimize the number of used bins. A bin is used if at least one item is assigned
to it.

Goal:
Return an assignment of items to bins and the minimum number of used bins.

QUBO convention:
Use binary assignment variables item-to-bin and binary used-bin variables.
Penalties enforce one bin per item and capacity limits.
