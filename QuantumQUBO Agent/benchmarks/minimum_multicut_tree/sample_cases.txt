MINIMUM MULTICUT ON TREES TEST CASES

Source inspiration:
William Cruz-Santos, Salvador E. Venegas-Andraca, and Marco Lanzagorta,
"A QUBO Formulation of Minimum Multicut Problem Instances in Trees for D-Wave
Quantum Annealers",
Scientific Reports 9, Article 17216, 2019.
DOI: https://doi.org/10.1038/s41598-019-53585-5

Benchmark simplification:
These are tiny weighted trees. A terminal pair is separated if at least one edge
on its unique tree path is cut.


========================================================================
TEST CASE 1: Path tree with two terminal pairs

Tree edges:
- E0: 0-1, cost 2
- E1: 1-2, cost 3
- E2: 2-3, cost 1

Terminal pairs to separate:
- 0, 2
- 1, 3

Ground truth:
- Minimum cut cost: 3
- Number of optimal cut sets: 2
- Optimal cut edge sets:
  - E1
  - E0, E2


========================================================================
TEST CASE 2: Star tree

Tree edges:
- E0: 0-1, cost 2
- E1: 0-2, cost 2
- E2: 0-3, cost 5
- E3: 0-4, cost 1

Terminal pairs to separate:
- 1, 2
- 3, 4

Ground truth:
- Minimum cut cost: 3
- Number of optimal cut sets: 2
- Optimal cut edge sets:
  - E0, E3
  - E1, E3
