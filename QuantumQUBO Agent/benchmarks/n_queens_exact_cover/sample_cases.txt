Benchmark simplification:
These cases use standard N-Queens feasibility instances kept at n=4 so the
natural row-column encoding (x_{r,c}) yields 16 binary variables, allowing
exact 2^n verification. The QUBO minimizes violations of row, column, and
diagonal constraints.

TEST CASE 1: Four queens on a 4x4 board

Board size: 4 x 4
Number of queens: 4

Ground truth:
- Minimum violation count: 0
- Number of optimal placements: 2
- Optimal placements:
  - (row 0, col 1), (row 1, col 3), (row 2, col 0), (row 3, col 2)
  - (row 0, col 2), (row 1, col 0), (row 2, col 3), (row 3, col 1)
