N-Queens Exact-Cover Problem

Given an N by N chessboard, place N queens so that no two queens attack each
other.

Task:
Choose queen positions.

Constraints:
- Exactly one queen in each row.
- Exactly one queen in each column.
- At most one queen on each diagonal.

Objective:
Find a feasible placement. Equivalently, minimize the number of violated row,
column, and diagonal constraints.

Goal:
Return queen coordinates and the minimum violation count.

QUBO convention:
Use one binary variable per board square. Quadratic penalties enforce row,
column, and diagonal constraints.
