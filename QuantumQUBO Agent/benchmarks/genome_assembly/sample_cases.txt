Benchmark note:
This genome-assembly-style overlap ordering case is inspired by QUBO sequence
ordering formulations and is hand curated.

Reads and their overlaps:
  r0 = ATG
  r1 = TGC
  r2 = GCA
  r3 = CAT

Overlap matrix O[i][j] (suffix of i matches prefix of j):
        r0  r1  r2  r3
  r0     0   2   1   0
  r1     0   0   2   1
  r2     1   0   0   2
  r3     2   1   0   0


This is a maximisation problem; QUBO minimisation objective value = -6.

Four optimal orderings (all achieve total overlap 6):
  Order [r0, r1, r2, r3]
  Order [r1, r2, r3, r0]
  Order [r2, r3, r0, r1]
  Order [r3, r0, r1, r2]

NB : Need to convert the ground truth ordering into binary variable appropriately for QUBO formulation.
