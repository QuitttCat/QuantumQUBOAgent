Cardinality-Constrained Portfolio Selection Problem

Given a set of assets, choose exactly K assets for a portfolio. Each asset has
an expected return score. Each pair of assets has a covariance/risk penalty.

Task:
Choose exactly K assets.

Objective:
Maximize the risk-adjusted score:

  total score =
    sum of expected return scores for selected assets
    - sum of pairwise risk penalties between selected assets.

Goal:
Return the selected assets and the maximum risk-adjusted score.

QUBO convention:
Use one binary variable per asset. The exactly-K cardinality constraint is
encoded as a quadratic penalty, and the objective is negated if a minimization
QUBO is required.
