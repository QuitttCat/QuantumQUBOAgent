Mars Satellite Observation Scheduling Problem

Given candidate observation windows for a Mars-orbiting instrument, choose a
subset of observations. Each observation has a science priority score and
resource usage. Some observations conflict because their time windows overlap or
because spacecraft/instrument resources cannot support both.

Task:
Choose a feasible set of observations.

Objective:
Maximize total science priority score.

Goal:
Return the selected observations and maximum total science score.

QUBO convention:
Use one binary variable per candidate observation. Linear terms reward science
priority. Quadratic penalties prohibit conflicting observation pairs and enforce
resource constraints.
