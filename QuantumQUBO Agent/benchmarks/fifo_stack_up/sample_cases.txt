FIFO STACK-UP TEST CASES

Source inspiration:
Colin Gebler, Jochen Rethmann, and Peer Ueberholz,
"QUBO Models for the FIFO Stack-Up Problem and Experimental Evaluation on a
Quantum Annealer",
SN Computer Science 5, Article 818, 2024.
DOI: https://doi.org/10.1007/s42979-024-03082-y

Benchmark simplification:
These tiny cases provide candidate output sequences and their open-pallet
profiles. The solver chooses the sequence with the smallest peak open count.


========================================================================
TEST CASE 1: Three pallets interleaved

Candidate output sequences:
- S0: A, B, A, C, B, C
  - open-pallet counts: 1, 2, 2, 3, 2, 1
  - maximum open pallets: 3
- S1: A, A, B, B, C, C
  - open-pallet counts: 1, 0, 1, 0, 1, 0
  - maximum open pallets: 1
- S2: A, B, C, A, B, C
  - open-pallet counts: 1, 2, 3, 3, 2, 1
  - maximum open pallets: 3

Ground truth:
- Minimum maximum open-pallet count: 1
- Number of optimal sequences: 1
- Selected sequence:
  - S1


========================================================================
TEST CASE 2: Two equivalent best sequences

Candidate output sequences:
- S0: A, B, A, B
  - open-pallet counts: 1, 2, 2, 1
  - maximum open pallets: 2
- S1: A, A, B, B
  - open-pallet counts: 1, 0, 1, 0
  - maximum open pallets: 1
- S2: B, B, A, A
  - open-pallet counts: 1, 0, 1, 0
  - maximum open pallets: 1

Ground truth:
- Minimum maximum open-pallet count: 1
- Number of optimal sequences: 2
- Selected sequences:
  - S1
  - S2
