FIFO Stack-Up Problem

Given a fixed incoming sequence of bins, each bin belongs to a pallet type. A
pallet is open between the first and last time a bin of that type is processed.

Task:
Choose an output order that respects the FIFO order of the input sequence.

Objective:
Minimize the maximum number of simultaneously open pallets.

Goal:
Return a valid output order and the minimum maximum open-pallet count.
