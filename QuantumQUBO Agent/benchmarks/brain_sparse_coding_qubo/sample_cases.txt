BRAIN-INSPIRED BINARY SPARSE CODING TEST CASES

Source inspiration:
Kyle Henke, Elijah Pelofske, Garrett Kenyon, and Georg Hahn,
"Comparing quantum annealing and spiking neuromorphic computing for sampling
binary sparse coding QUBO problems",
npj Unconventional Computing 2, Article 13, 2025.
DOI: https://doi.org/10.1038/s44335-025-00028-2

Benchmark simplification:
The paper studies binary sparse coding, a brain-inspired representation-learning
problem, as a QUBO and compares quantum annealing with spiking neuromorphic
sampling. These cases provide small dictionary atoms with precomputed sparse
coding energies.


========================================================================
TEST CASE 1: Three atoms, choose a sparse reconstruction

Dictionary atoms:
- A0_vertical_edge
- A1_horizontal_edge
- A2_noise_patch

Candidate selections:
- S0: select none
  - reconstruction error: 6
  - sparsity penalty: 0
  - total energy: 6
- S1: select A0_vertical_edge
  - reconstruction error: 2
  - sparsity penalty: 1
  - total energy: 3
- S2: select A1_horizontal_edge
  - reconstruction error: 3
  - sparsity penalty: 1
  - total energy: 4
- S3: select A0_vertical_edge and A1_horizontal_edge
  - reconstruction error: 0
  - sparsity penalty: 2
  - total energy: 2
- S4: select A0_vertical_edge and A2_noise_patch
  - reconstruction error: 3
  - sparsity penalty: 2
  - total energy: 5

Ground truth:
- Minimum sparse-coding energy: 2
- Number of optimal selections: 1
- Selected dictionary atoms:
  - A0_vertical_edge
  - A1_horizontal_edge


========================================================================
TEST CASE 2: Redundant atoms compete under sparsity pressure

Dictionary atoms:
- A0_left_stroke
- A1_right_stroke
- A2_full_bar
- A3_background_texture

Candidate selections:
- S0: select A0_left_stroke and A1_right_stroke
  - reconstruction error: 1
  - sparsity penalty: 2
  - total energy: 3
- S1: select A2_full_bar
  - reconstruction error: 2
  - sparsity penalty: 1
  - total energy: 3
- S2: select A0_left_stroke, A1_right_stroke, and A3_background_texture
  - reconstruction error: 0
  - sparsity penalty: 3
  - total energy: 3
- S3: select A3_background_texture
  - reconstruction error: 7
  - sparsity penalty: 1
  - total energy: 8

Ground truth:
- Minimum sparse-coding energy: 3
- Number of optimal selections: 3
- Selected atom sets:
  - A0_left_stroke, A1_right_stroke
  - A2_full_bar
  - A0_left_stroke, A1_right_stroke, A3_background_texture
