Remote-Sensing Image Segmentation Problem

Given a remote-sensing image represented as a grid graph, assign each pixel to
one of two segments. Unary costs represent pixel-label preference and pairwise
costs represent boundary penalties between neighboring pixels.

Task:
Assign one segment label to each pixel.

Objective:
Minimize segmentation energy.

Goal:
Return the selected segmentation and minimum energy.

QUBO convention:
Use one binary variable per pixel. Unary terms encode label costs and quadratic
terms encode graph-cut boundary penalties.
