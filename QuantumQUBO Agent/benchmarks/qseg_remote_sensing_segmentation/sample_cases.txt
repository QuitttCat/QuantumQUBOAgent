Q-SEG REMOTE-SENSING SEGMENTATION TEST CASES

Source inspiration:
Supreeth Mysore Venkatesh, Antonio Macaluso, Marlon Nuske, Matthias Klusch,
and Andreas Dengel,
"Q-Seg: Quantum Annealing-Based Unsupervised Image Segmentation",
IEEE Computer Graphics and Applications, 2024.
DOI: https://doi.org/10.1109/MCG.2024.3455012

GitHub implementation:
Supreeth Mysore Venkatesh,
"Q-Seg: Quantum Annealing-based Unsupervised Segmentation Algorithm",
GitHub repository.
URL: https://github.com/supreethmv/Q-Seg

Benchmark simplification:
Q-Seg converts images into weighted grid graphs and solves a QUBO/min-cut style
segmentation problem on a quantum annealer. These cases use tiny remote-sensing
style grids with precomputed segmentation energies.
