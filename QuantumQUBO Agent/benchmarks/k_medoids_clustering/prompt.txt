K-Medoids Prototype Selection Problem

Given N objects and pairwise distances, choose exactly K objects as medoids.
Every object is assigned to its nearest selected medoid.

Task:
Choose exactly K medoids.

Objective:
Minimize:

  total assignment distance =
    sum over all objects of distance to the nearest selected medoid.

Goal:
Return the selected medoids and the minimum total assignment distance.

QUBO convention:
Use one binary variable per candidate medoid, with a quadratic penalty enforcing
exactly K selected medoids. The nearest-medoid assignment cost is precomputed
for each selected medoid subset in these small benchmark cases.
