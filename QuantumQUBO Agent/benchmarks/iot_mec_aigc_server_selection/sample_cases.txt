IOT MEC AIGC SERVER SELECTION TEST CASES

Source inspiration:
Wenrui Zhang, Changshi Zhou, Weiqi Liu, Tao Han, and Nirwan Ansari,
"Hybrid Quantum-Inspired Optimization for AIGC-Driven IoT Task Offloading in
MEC Networks",
IEEE Internet of Things Journal 13(7), pp. 14477-14485, 2026.
DOI: https://doi.org/10.1109/JIOT.2025.3649393

Benchmark simplification:
The paper combines wireless bandwidth preallocation with QUBO-based AIGC server
selection for IoT devices connected to mobile edge computing servers. These
cases keep the QUBO core: assign each IoT task to one server while minimizing
latency/bandwidth cost under server capacities.


========================================================================
TEST CASE 1: Three IoT tasks, two MEC servers

IoT tasks:
- T0_camera_summary
- T1_audio_caption
- T2_sensor_report

MEC servers:
- E0:
  - capacity: 2 tasks
- E1:
  - capacity: 2 tasks

Assignment costs:
- T0_camera_summary: E0=2, E1=5
- T1_audio_caption: E0=3, E1=2
- T2_sensor_report: E0=4, E1=1

Ground truth:
- Minimum total assignment cost: 5
- Number of optimal assignments: 1
- Assignments:
  - T0_camera_summary -> E0
  - T1_audio_caption -> E1
  - T2_sensor_report -> E1
- Score calculation:
  - 2 + 2 + 1 = 5


========================================================================
TEST CASE 2: Capacity-limited edge selection

IoT tasks:
- T0_image_generation
- T1_text_generation
- T2_video_caption
- T3_anomaly_report

MEC servers:
- E0:
  - capacity: 2 tasks
- E1:
  - capacity: 1 task
- E2:
  - capacity: 2 tasks

Assignment costs:
- T0_image_generation: E0=2, E1=4, E2=5
- T1_text_generation: E0=3, E1=1, E2=4
- T2_video_caption: E0=4, E1=3, E2=2
- T3_anomaly_report: E0=2, E1=5, E2=3

Ground truth:
- Minimum total assignment cost: 7
- Number of optimal assignments: 1
- Assignments:
  - T0_image_generation -> E0
  - T1_text_generation -> E1
  - T2_video_caption -> E2
  - T3_anomaly_report -> E0
- Score calculation:
  - 2 + 1 + 2 + 2 = 7
