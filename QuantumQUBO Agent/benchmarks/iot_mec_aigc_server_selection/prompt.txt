IoT MEC AIGC Server Selection Problem

Given IoT devices with AI-generated-content tasks and candidate mobile edge
computing servers, select one server for each task. Each assignment has a
latency cost and servers have limited capacity. Some assignments may also incur
bandwidth penalties.

Task:
Assign each IoT task to exactly one edge server.

Constraints:
- Every task is assigned exactly once.
- Server capacity limits must be respected.

Objective:
Minimize total latency and bandwidth cost.

Goal:
Return the task-server assignment and minimum total cost.

QUBO convention:
Use one binary variable per task-server assignment. One-hot penalties enforce
single-server assignment, capacity penalties discourage overloaded servers, and
linear terms encode latency and bandwidth costs.
