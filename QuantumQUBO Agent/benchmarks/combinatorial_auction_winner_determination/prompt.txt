Combinatorial Auction Winner Determination Problem

Given items and bids on bundles of items, choose winning bids. No item may be
allocated to more than one winning bid.

Task:
Select a feasible set of non-overlapping bids.

Objective:
Maximize total winning-bid revenue.

Goal:
Return the selected bids and maximum revenue.

QUBO convention:
Use one binary variable per bid. Quadratic penalties prohibit selecting two bids
that share an item.
