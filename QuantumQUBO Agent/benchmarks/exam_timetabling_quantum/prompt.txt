Exam Timetabling Problem

Given exams, time slots, and student-conflict constraints, assign every exam to
one time slot.

Task:
Schedule all exams.

Constraints:
- Each exam must be assigned to exactly one slot.
- Exams sharing students should not be assigned to the same slot.

Objective:
Minimize the number of student conflicts, with optional soft penalties for
undesirable slot choices.

Goal:
Return the exam-to-slot schedule and minimum conflict penalty.

QUBO convention:
Use one binary variable per exam-slot assignment. One-hot penalties enforce one
slot per exam, and quadratic penalties discourage conflicting exams in the same
slot.
