EXAM TIMETABLING TEST CASES

Source inspiration:
"Hybrid quantum annealing for large-scale exam scheduling: Validation in
real-world educational scenarios",
Applied Soft Computing, 2025.
DOI: https://doi.org/10.1016/j.asoc.2025.113756

Benchmark simplification:
The source formulates large-scale exam scheduling as a QUBO and solves it using
hybrid quantum annealing with graph-based preprocessing. These cases use small
exam-conflict graphs and discrete time slots.


========================================================================
TEST CASE 1: Three exams, two slots

Exams: E0, E1, E2
Time slots: S0, S1

Student-conflict pairs:
- E0 conflicts with E1
- E1 conflicts with E2

Ground truth:
- Minimum conflict count: 0
- Number of optimal schedules: 2
- Optimal schedules:
  - E0 -> S0, E1 -> S1, E2 -> S0
  - E0 -> S1, E1 -> S0, E2 -> S1


========================================================================
TEST CASE 2: Triangle conflict graph with two slots

Exams: E0, E1, E2
Time slots: S0, S1

Student-conflict pairs:
- E0 conflicts with E1
- E1 conflicts with E2
- E0 conflicts with E2

Ground truth:
- Minimum conflict count: 1
- Number of optimal schedules: 6
- One optimal schedule:
  - E0 -> S0
  - E1 -> S1
  - E2 -> S0


========================================================================
TEST CASE 3: Four exams, two independent conflict chains

Exams: E0, E1, E2, E3
Time slots: S0, S1

Student-conflict pairs:
- E0 conflicts with E1
- E2 conflicts with E3
- E1 conflicts with E2

Ground truth:
- Minimum conflict count: 0
- Number of optimal schedules: 2
- Optimal schedules:
  - E0 -> S0, E1 -> S1, E2 -> S0, E3 -> S1
  - E0 -> S1, E1 -> S0, E2 -> S1, E3 -> S0
