Core-Periphery Partitioning Problem

Given an undirected network, assign each node to either the core or the
periphery. A good core-periphery partition places densely connected nodes in the
core and sparsely connected nodes in the periphery.

Task:
Choose a binary label for every node:
- 1 means core,
- 0 means periphery.

Objective:
Maximize the supplied core-periphery quality score.

Goal:
Return the node labels and maximum quality score.

QUBO convention:
Use one binary variable per node. Quadratic terms score pairwise node
relationships according to a core-periphery kernel. If using minimization,
minimize the negative quality score.
