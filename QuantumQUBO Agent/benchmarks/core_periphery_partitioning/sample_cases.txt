CORE-PERIPHERY PARTITIONING TEST CASES

Source inspiration:
Catherine F. Higham, Desmond J. Higham, and Francesco Tudisco,
"Core-periphery Partitioning and Quantum Annealing",
Proceedings of the 28th ACM SIGKDD Conference on Knowledge Discovery and Data
Mining, KDD 2022, pp. 565-573.
DOI: https://doi.org/10.1145/3534678.3539261



Benchmark simplification:
The KDD paper defines a core-periphery quality kernel whose optimization can be
expressed as a QUBO and solved with quantum annealing. These benchmark cases use
small networks and list candidate partitions with precomputed quality scores.


========================================================================
TEST CASE 1: Four-node network with two-node core

Nodes: 0, 1, 2, 3

Edges:
- 0, 1
- 0, 2
- 1, 2
- 0, 3

Candidate partitions:
- P0:
  - core nodes: 0, 1
  - periphery nodes: 2, 3
  - quality score: 7
- P1:
  - core nodes: 0, 1, 2
  - periphery nodes: 3
  - quality score: 9
- P2:
  - core nodes: 2, 3
  - periphery nodes: 0, 1
  - quality score: 2

Ground truth:
- Maximum core-periphery quality score: 9
- QUBO minimization objective value: -9
- Number of optimal partitions: 1
- Selected partition:
  - P1
