Single-Track Railway Dispatching Problem

Given trains that need to pass through a shared single-track segment, choose one
timing option for each train. Conflicting options cannot be selected together.
Each option has a delay cost.

Task:
Choose exactly one option for every train.

Objective:
Minimize:

  total dispatch cost =
    sum of selected option delay costs
    + large penalties for selected conflicting option pairs.

Goal:
Return the selected timing option for each train and the minimum feasible delay
cost.
