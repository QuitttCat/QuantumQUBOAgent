QUBO Benchmark Job Selection Problem

Given candidate benchmark jobs, choose a subset to run under a compute-time
budget. Each job covers a problem family and solver type. The goal is to cover
as many benchmark categories as possible while respecting the budget.

Task:
Choose benchmark jobs.

Objective:
Maximize coverage value minus duplicate-coverage penalties.

Goal:
Return selected jobs and maximum benchmark value.

QUBO convention:
Use one binary variable per candidate job. Linear terms reward benchmark value,
quadratic terms penalize redundant coverage, and a capacity penalty enforces the
runtime budget.
