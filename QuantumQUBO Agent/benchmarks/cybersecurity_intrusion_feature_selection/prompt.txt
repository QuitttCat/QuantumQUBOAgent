Network Intrusion Feature Selection Problem

Given candidate network-traffic features, choose exactly K features for an
intrusion detection model. Each feature has a detection relevance score and
pairs of features have redundancy penalties.

Task:
Choose exactly K features.

Objective:
Maximize:

  total score =
    sum of selected detection relevance scores
    - sum of selected pairwise redundancy penalties.

Goal:
Return the selected features and maximum total score.
