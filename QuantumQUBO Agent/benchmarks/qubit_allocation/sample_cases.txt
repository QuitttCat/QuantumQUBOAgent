QUBIT ALLOCATION TEST CASES

Source inspiration:
Bryan Dury and Olivia Di Matteo,
"A QUBO Formulation for Qubit Allocation",
arXiv:2009.00140, 2020.
DOI: https://doi.org/10.48550/arXiv.2009.00140

Benchmark simplification:
These cases use small logical circuits and precomputed physical pair costs. The
QUBO structure is a one-hot assignment/permutation problem with pairwise gate
costs.


========================================================================
TEST CASE 1: Three logical qubits on a line device

Logical qubits: q0, q1, q2
Physical qubits: p0, p1, p2

Two-qubit logical gates:
- q0, q1
- q1, q2

Physical pair costs:
- p0, p1: 1
- p1, p2: 1
- p0, p2: 3

Ground truth:
- Minimum allocation cost: 2
- Number of optimal mappings: 2
- Optimal mappings:
  - q0->p0, q1->p1, q2->p2
  - q0->p2, q1->p1, q2->p0


========================================================================
TEST CASE 2: High-fidelity edge preference

Logical qubits: q0, q1, q2
Physical qubits: p0, p1, p2

Two-qubit logical gates:
- q0, q1
- q0, q2

Physical pair costs:
- p0, p1: 1
- p0, p2: 1
- p1, p2: 4

Ground truth:
- Minimum allocation cost: 2
- Number of optimal mappings: 2
- Optimal mappings:
  - q0->p0, q1->p1, q2->p2
  - q0->p0, q1->p2, q2->p1
