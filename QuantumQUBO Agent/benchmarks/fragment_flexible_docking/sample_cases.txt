FRAGMENT-BASED FLEXIBLE DOCKING TEST CASES

Source inspiration:
Keisuke Yanagisawa, Takuya Fujie, Kazuki Takabatake, and Yutaka Akiyama,
"QUBO Problem Formulation of Fragment-Based Protein-Ligand Flexible Docking",
Entropy 2024, 26(5), 397.
DOI: https://doi.org/10.3390/e26050397

Benchmark simplification:
These are small synthetic placement-selection cases inspired by the paper's
fragment-based docking formulation. Each candidate keeps its fragment ID,
placement ID, and interaction energy. Each pairwise term keeps both placement
IDs, a clash penalty, and a bond reward. The objective is to minimize:

  selected placement energies + selected clash penalties - selected bond rewards.

Each fragment must select exactly one placement.

TEST CASE 1: 2 fragments, compatible bonded placements

Fragments:
- F0
- F1

- Fragment F0:
  - Placement F0_P0:
    - interaction energy -4
  - Placement F0_P1:
    - interaction energy -1
- Fragment F1:
  - Placement F1_P0:
    - interaction energy -3
  - Placement F1_P1:
    - interaction energy -5

Pairwise placement terms:
- Pair F0_P0 with F1_P0:
  - clash penalty 0
  - bond reward 2
- Pair F0_P0 with F1_P1:
  - clash penalty 4
  - bond reward 0
- Pair F0_P1 with F1_P0:
  - clash penalty 0
  - bond reward 1
- Pair F0_P1 with F1_P1:
  - clash penalty 0
  - bond reward 1

Ground truth:
- Minimum total docking score: -9
- Number of optimal solutions: 1
- Selected placements:
  - fragment F0 -> F0_P0
  - fragment F1 -> F1_P0
- Score calculation:
  - selected energies: -4 + -3 = -7
  - selected clash penalties: 0
  - selected bond rewards: 2
  - total score: -7 + 0 - 2 = -9
