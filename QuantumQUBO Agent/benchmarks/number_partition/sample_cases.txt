Test case 1:
N = 8
S = {25, 7, 13, 31, 42, 17, 21, 10}

An optimal assignment is:
x = (0, 0, 0, 1, 1, 0, 0, 1) or x = (1, 1, 1, 0, 0, 1, 1, 0)

This gives a subset = {31, 42, 10} with sum 83 and other subset = {25, 7, 13, 17, 21}
with sum 83, so the difference is 0 (perfect partition).

Test case 2: 
N = 10
S = {4, 5, 2, 10, 6, 11, 9, 4, 9, 1}

An optimal assignment is:
x = (0, 0, 0, 0, 0, 1, 1, 0, 1, 1) or x = (1, 1, 1, 1, 1, 0, 0, 1, 0, 0)
