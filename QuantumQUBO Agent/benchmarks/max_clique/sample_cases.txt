Test case 1: 4 nodes, diamond graph (K4 minus one edge)

Source: Classic graph theory textbook example

Number of vertices: 4
Edges:
  (0, 1), (0, 2), (0, 3), (1, 2), (1, 3)

Note: Edge (2, 3) is absent. Vertices 0 and 1 are the hub pair;
      vertices 2 and 3 are not connected to each other.

Maximum clique size = 3

Optimal cliques (two solutions):
  {0, 1, 2} ,
  {0, 1, 3} 

QUBO optimal value = -3

Test case 2: 5 nodes, cycle with chord

Source: Custom

Number of vertices: 5
Edges:
  (0, 1), (1, 2), (2, 3), (3, 4), (4, 0), (0, 2)

Note: C5 cycle with chord (0, 2), creating triangle {0, 1, 2}.
      No other triangle exists in this graph.

Maximum clique size = 3

Optimal clique (unique solution):
  {0, 1, 2}  

QUBO optimal value = -3



========================================================================
Test case 3: 6 nodes, K4 subgraph with two peripheral vertices

Source: Inspired by the six-node DNA computing example in Ouyang et al. (1997)
        and https://cs.stanford.edu/people/eroberts/courses/soco/projects/2003-04/dna-computing/clique.htm

Number of vertices: 6
Edges:
  (2, 3), (2, 4), (2, 5), (3, 4), (3, 5), (4, 5),
  (0, 2), (0, 3), (1, 4), (1, 5)

Note: Vertices {2, 3, 4, 5} form a complete subgraph K4.
      Vertex 0 connects only to 2 and 3 (not 4, 5).
      Vertex 1 connects only to 4 and 5 (not 2, 3).
      Vertices 0 and 1 are not connected to each other.

Maximum clique size = 4

Optimal clique (unique solution):
  {2, 3, 4, 5}  

QUBO optimal value = -4



========================================================================
Test case 4: 7 nodes, circulant graph C(7, {1, 2})

Source: Custom , circulant graph where each vertex connects to its two
        nearest neighbours in each direction on a cycle.

Number of vertices: 7
Edges:
  (0, 1), (0, 2), (1, 2), (1, 3), (2, 3), (2, 4),
  (3, 4), (3, 5), (4, 5), (4, 6), (5, 6), (5, 0),
  (6, 0), (6, 1)

Note: Each vertex i is adjacent to i±1 and i±2 (mod 7), giving a 4-regular graph.
      Every pair of adjacent cycle-neighbours forms a triangle.
      No K4 exists (no four mutually adjacent vertices).

Maximum clique size = 3

Optimal cliques (seven solutions, one per consecutive triple on the cycle):
  {0, 1, 2}  
  {1, 2, 3} 
  {2, 3, 4}  
  {3, 4, 5}  
  {4, 5, 6}  
  {0, 5, 6}  
  {0, 1, 6} 
QUBO optimal value = -3
