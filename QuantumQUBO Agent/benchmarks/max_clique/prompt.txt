Maximum Clique Problem

A clique in an undirected graph is a subset of vertices in which every pair of vertices
is directly connected by an edge — every vertex in the subset is adjacent to every other.

The Maximum Clique Problem asks: given an undirected graph G = (V, E), find the largest
clique, i.e., the subset C ⊆ V of maximum size such that all pairs within C are connected.

Constraint type note:
- Pairwise inequality x_i + x_j <= 1 (binary vars)  →  penalty * x_i * x_j  (product only — NOT (x_i+x_j-1)^2 which is an equality penalty and penalizes (0,0) pairs, breaking the formulation)
- CRITICAL: (x_i + x_j - 1)^2 = 1 for (0,0) AND (1,1) — it is an EQUALITY penalty, NOT an inequality. NEVER use it to encode x_i + x_j <= 1.
