Benchmark simplification:
Each car has two or three candidate routes. The congestion score counts how
many selected-route pairs overlap on each road segment.


========================================================================
TEST CASE 1: Three cars, two routes each

Cars: C0, C1, C2

Candidate routes:
- C0_R0: segments A, B
- C0_R1: segments C
- C1_R0: segments A, D
- C1_R1: segments B, E
- C2_R0: segments C, D
- C2_R1: segments E

Ground truth:
- Minimum congestion score: 0
- Number of optimal assignments: 2
- Optimal assignments:
  - C0 -> C0_R1, C1 -> C1_R1, C2 -> C2_R0
  - C0 -> C0_R1, C1 -> C1_R0, C2 -> C2_R1


========================================================================
TEST CASE 2: Four cars, unavoidable overlap

Cars: C0, C1, C2, C3

Candidate routes:
- C0_R0: segments A, B
- C0_R1: segments C
- C1_R0: segments A
- C1_R1: segments B, D
- C2_R0: segments C, D
- C2_R1: segments A
- C3_R0: segments B
- C3_R1: segments C, D

Ground truth:
- Minimum congestion score: 1
- One optimal assignment:
  - C0 -> C0_R1
  - C1 -> C1_R1
  - C2 -> C2_R1
  - C3 -> C3_R0
- Score calculation:
  - segment B is shared by C1_R1 and C3_R0: 1 pair
  - total overlap cost: 1
