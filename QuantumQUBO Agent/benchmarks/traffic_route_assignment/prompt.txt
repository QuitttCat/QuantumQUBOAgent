Traffic Route Assignment Problem

Given cars and a small set of candidate routes for each car, choose exactly one
route per car. A route is represented by the road segments it uses. Congestion
cost is incurred when multiple selected routes use the same segment.

Task:
Assign one route to every car.

Objective:
Minimize total overlap cost:

  total overlap cost =
    sum over road segments of number of selected-route pairs sharing that segment.

Goal:
Return the chosen route for each car and the minimum congestion score.
