DYNAMIC SPECTRUM ALLOCATION TEST CASES

Source inspiration:
Kazuhiro Saito, Akio Ikami, and Chihiro Ono,
"Evaluating dynamic spectrum allocation using quantum annealing",
IEICE Communications Express 10(9), pp. 726-732, 2021.
DOI: https://doi.org/10.1587/comex.2021XBL0047

Yuhi Kurebayashi, Yui Yamashita, and Yoshito Tobe,
"Optimization of Penalty Weights in Quantum Annealing for Dynamic Spectrum
Allocation",
IEICE Proceeding Series 97, 2025.
DOI: https://doi.org/10.34385/proc.97.T-2-2-1

Benchmark simplification:
These cases use small link-channel assignment tables. Interference penalties
apply only when listed interfering links choose the same channel.


========================================================================
TEST CASE 1: Three links, two shared channels

Links: L0, L1, L2
Channels: C0, C1

Assignment utilities:
- L0: C0=5, C1=3
- L1: C0=4, C1=4
- L2: C0=3, C1=5

Interfering link pairs:
- L0, L1: same-channel penalty 4
- L1, L2: same-channel penalty 3

Ground truth:
- Maximum allocation score: 12
- QUBO minimization objective value: -12
- Number of optimal assignments: 1
- Assignments:
  - L0 -> C0
  - L1 -> C1
  - L2 -> C0
- Score calculation:
  - utility: 5 + 4 + 3 = 12
  - interference penalties: 0
  - total score: 12


========================================================================
TEST CASE 2: Priority link competes with neighborhood interference

Links: L0, L1, L2, L3
Channels: C0, C1

Assignment utilities:
- L0: C0=7, C1=4
- L1: C0=5, C1=6
- L2: C0=4, C1=5
- L3: C0=6, C1=3

Interfering link pairs:
- L0, L1: same-channel penalty 5
- L0, L2: same-channel penalty 4
- L1, L3: same-channel penalty 3

Ground truth:
- Maximum allocation score: 24
- QUBO minimization objective value: -24
- Number of optimal assignments: 1
- Assignments:
  - L0 -> C0
  - L1 -> C1
  - L2 -> C1
  - L3 -> C0
- Score calculation:
  - utility: 7 + 6 + 5 + 6 = 24
  - interference penalties: 0
  - total score: 24
