Dynamic Spectrum Allocation Problem

Given radio links and available channels, assign exactly one channel to each
link. Some pairs of links interfere if they use the same channel. Each
link-channel assignment can also have a utility score.

Task:
Assign one channel to every radio link.

Objective:
Maximize total utility minus interference penalties.

Goal:
Return the channel assignment and maximum score.

QUBO convention:
Use one binary variable per link-channel pair. One-hot penalties enforce one
channel per link, and quadratic penalties discourage interfering links from
using the same channel.
