LLM Cascade Routing Problem

Given a set of user requests and a cascade of available language models, assign
each request to exactly one model tier. Each assignment has an inference cost
and a predicted quality score. Some requests require minimum quality.

Task:
Choose one model for every request.

Constraints:
- Each request must be routed to exactly one model.
- The selected model must satisfy the request quality requirement.

Objective:
Minimize total inference cost while satisfying quality requirements.

Goal:
Return the request-model assignment and minimum total cost.

QUBO convention:
Use one binary variable per request-model assignment. One-hot penalties enforce
one model per request. Linear terms encode cost, and penalties exclude
assignments that fail the quality threshold.
