Ligand Pose Hotspot Sampling Problem

In structure-based drug discovery, a small molecule ligand must be placed inside a
protein binding pocket in a physically plausible way. One useful first step is to identify
promising hotspot locations inside the pocket where ligand atoms could be placed.

The binding pocket is represented as a finite set of 3D grid points. Each grid point has
an interaction energy score that estimates how favorable it would be to place a ligand atom
at that location. Lower, more negative scores are more attractive. Higher or positive scores
are less favorable.

The selected grid points should also be spatially reasonable. Points that are extremely
close to each other should not both be selected, because that would represent overlapping
or clashing ligand atoms. Points separated by a preferred distance range are desirable,
because they can form a well-spread geometric pattern for later ligand placement.

Input:
- A set of grid points inside a protein binding pocket.
- For each grid point:
  - a 3D coordinate,
  - an interaction energy score.
- A minimum clash distance. Two selected grid points closer than this distance create
  a steric clash and should be discouraged.
- A preferred distance range. Pairs of selected grid points whose distance falls in this
  range should be encouraged.
- A clash penalty strength.
- A preferred-spacing reward strength.

Task:
Select a subset of grid points to use as ligand-placement hotspots.

Objective:
Find a subset of grid points that has low total interaction energy, avoids selecting pairs
of points that are too close together, and favors pairs of points whose separation lies in
the preferred distance range.

Goal:
Return the selected hotspot grid points that best balance energetic attraction, clash
avoidance, and spatial distribution.
