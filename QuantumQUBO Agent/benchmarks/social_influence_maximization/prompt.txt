Social Influence Maximization Problem

Given a social network and an influence-reach table for each candidate seed
node, choose exactly K seed nodes to maximize the number of influenced users.

Task:
Choose K seed nodes.

Objective:
Maximize total covered influence reach, avoiding double-counting users reached
by multiple selected seeds.

Goal:
Return the selected seed nodes and maximum influence coverage.

QUBO convention:
Influence maximization can be converted to a maximum coverage formulation. Use
binary seed variables and auxiliary coverage variables, or a reduced QUBO with
pairwise overlap penalties between seed nodes.
