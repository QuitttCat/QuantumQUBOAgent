ALKANE STRUCTURAL ISOMER SEARCH TEST CASES

Source inspiration:
Jason P. Terry, Prosper D. Akrobotu, Christian F. A. Negre, and Susan M.
Mniszewski,
"Quantum isomer search",
PLOS ONE 15(1), e0226787, 2020.
DOI: https://doi.org/10.1371/journal.pone.0226787

Benchmark simplification:
The full paper formulates alkane structural isomer enumeration as a QUBO. These
small cases list candidate carbon-carbon skeletons directly and ask for the
valid isomer classes.


========================================================================
TEST CASE 1: Butane skeletons

Formula: C4H10

Candidate skeletons:
- S0: chain 0-1-2-3
- S1: branched center 0 connected to 1, 2, and 3
- S2: cycle 0-1-2-3-0

Validity rules:
- Must be connected.
- Must be acyclic.
- Carbon degree must be at most 4.
- For C4H10, there must be exactly 3 carbon-carbon bonds.

Ground truth:
- Number of valid skeletons in this candidate list: 2
- Valid skeletons:
  - S0
  - S1


========================================================================
TEST CASE 2: Pentane candidate classes

Formula: C5H12

Candidate skeletons:
- S0: chain 0-1-2-3-4
- S1: one branch, edges 0-1, 1-2, 2-3, 2-4
- S2: central carbon connected to 1, 2, 3, 4
- S3: cycle 0-1-2-3-4-0

Validity rules:
- Must be connected.
- Must be acyclic.
- Carbon degree must be at most 4.
- For C5H12, there must be exactly 4 carbon-carbon bonds.

Ground truth:
- Number of valid skeletons in this candidate list: 3
- Valid skeletons:
  - S0
  - S1
  - S2
