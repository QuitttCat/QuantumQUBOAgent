Mutual-Information Sensor Placement Problem

Given candidate sensor locations, choose exactly K locations. Each location has
an information score, and pairs of locations have redundancy penalties when
their measurements are strongly correlated.

Task:
Choose K sensor positions.

Objective:
Maximize:

  total information score =
    sum of selected individual information scores
    - sum of selected pairwise redundancy penalties.

Goal:
Return selected sensor locations and maximum total information score.

QUBO convention:
Use one binary variable per candidate sensor. A quadratic penalty enforces
exactly K selected sensors. Pairwise terms approximate mutual-information
redundancy between selected sensor positions.
