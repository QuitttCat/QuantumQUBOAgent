Balanced Distance-Based Clustering Problem

Given N data points and a target number of clusters, assign each point to one
cluster. Each cluster must contain the specified number of points. Pairwise
distances are provided directly.

For this benchmark, the objective is to minimize the sum of distances between
pairs of points assigned to the same cluster.

Task:
Assign every point to exactly one cluster while satisfying the cluster-size
requirements.

Objective:
Minimize:

  total within-cluster distance =
    sum of distances for all point pairs that share a cluster.

Goal:
Return the cluster assignment and the minimum total within-cluster distance.
