GRAPH ISOMORPHISM TEST CASES

Source inspiration:
Hristo N. Djidjev, Guillaume Chapuis, Georg Hahn, and Guillaume Rizk,
"QUBO formulations for the graph isomorphism problem and related problems",
Theoretical Computer Science 701, pp. 54-69, 2017.
DOI: https://doi.org/10.1016/j.tcs.2017.04.016

Benchmark simplification:
These cases use tiny graphs and report the mismatch count under the best
bijection. The one-hot permutation constraints are the main QUBO structure.


========================================================================
TEST CASE 1: Two isomorphic paths on four vertices

Graph G edges:
- 0, 1
- 1, 2
- 2, 3

Graph H edges:
- A, C
- C, D
- D, B

Ground truth:
- Minimum adjacency mismatches: 0
- Number of isomorphisms: 2
- One valid mapping:
  - 0 -> A
  - 1 -> C
  - 2 -> D
  - 3 -> B


========================================================================
TEST CASE 2: Path versus star, not isomorphic

Graph G edges:
- 0, 1
- 1, 2
- 2, 3

Graph H edges:
- A, B
- A, C
- A, D

Ground truth:
- Minimum adjacency mismatches: 2
- Number of exact isomorphisms: 0
- One best mapping:
  - 0 -> B
  - 1 -> A
  - 2 -> C
  - 3 -> D
