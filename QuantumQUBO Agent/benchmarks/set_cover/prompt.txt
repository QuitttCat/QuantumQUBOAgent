Weighted Set Cover Problem

Given a universe of elements and a collection of weighted subsets, choose
subsets so that every universe element is covered at least once.

Task:
Choose a minimum-cost family of subsets that covers the universe.

Objective:
Minimize the total cost of selected subsets.

Goal:
Return the selected subsets and the minimum total cost.

QUBO convention:
Use one binary variable per subset. Coverage constraints are encoded with
penalties, often requiring slack or inequality-handling terms in a full QUBO.
