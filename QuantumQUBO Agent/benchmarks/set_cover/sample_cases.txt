WEIGHTED SET COVER TEST CASES

Source inspiration:
Yudong Cao, Shuxian Jiang, Debbie Perouli, and Sabre Kais,
"Solving Set Cover with Pairs Problem using Quantum Annealing",
Scientific Reports 6, Article 33957, 2016.
DOI: https://doi.org/10.1038/srep33957

Hristo N. Djidjev,
"Quantum Annealing with Inequality Constraints: The Set Cover Problem",
Advanced Quantum Technologies 6(11), 2300104, 2023.
DOI: https://doi.org/10.1002/qute.202300104

Benchmark simplification:
These are ordinary weighted set cover instances rather than set-cover-with-pairs
instances. The focus is on coverage constraints and subset-selection cost.


========================================================================
TEST CASE 1: Four elements, three sets

Universe: U0, U1, U2, U3

Candidate sets:
- S0: cost 3, covers U0, U1
- S1: cost 2, covers U1, U2
- S2: cost 3, covers U2, U3
- S3: cost 5, covers U0, U3

Ground truth:
- Minimum total cost: 6
- Number of optimal covers: 1
- Selected sets:
  - S0
  - S2
- Coverage:
  - U0 by S0
  - U1 by S0
  - U2 by S2
  - U3 by S2


========================================================================
TEST CASE 2: Five elements with a cheap overlapping cover

Universe: U0, U1, U2, U3, U4

Candidate sets:
- S0: cost 4, covers U0, U1, U2
- S1: cost 2, covers U0, U3
- S2: cost 2, covers U1, U4
- S3: cost 3, covers U2, U3, U4
- S4: cost 5, covers U0, U4

Ground truth:
- Minimum total cost: 7
- Number of optimal covers: 1
- Selected sets:
  - S0
  - S3
- Coverage:
  - S0 covers U0, U1, U2
  - S3 covers U2, U3, U4
