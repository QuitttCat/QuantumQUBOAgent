MOLECULAR CONFORMATION SELECTION TEST CASES

Source inspiration:
D. J. J. Marchand et al.,
"A Variable Neighbourhood Descent Heuristic for Conformational Search Using a
Quantum Annealer",
Scientific Reports 9, Article 13708, 2019.
DOI: https://doi.org/10.1038/s41598-019-47298-y

Sami Boulebnane et al.,
"Peptide conformational sampling using the Quantum Approximate Optimization
Algorithm",
npj Quantum Information 9, Article 70, 2023.
DOI: https://doi.org/10.1038/s41534-023-00733-5

Benchmark simplification:
The instances use a small discrete rotamer library and pairwise interaction
table. This captures the one-hot and pair-energy structure of QUBO
conformational search.


========================================================================
TEST CASE 1: Two torsions, three rotamers each

Torsions: T0, T1

Local energies:
- T0_R0: 0
- T0_R1: 2
- T0_R2: 1
- T1_R0: 1
- T1_R1: 0
- T1_R2: 3

Pairwise interaction terms:
- T0_R0, T1_R0: 2
- T0_R0, T1_R1: 0
- T0_R0, T1_R2: 1
- T0_R1, T1_R0: -1
- T0_R1, T1_R1: 1
- T0_R1, T1_R2: 0
- T0_R2, T1_R0: 0
- T0_R2, T1_R1: -2
- T0_R2, T1_R2: 1

Ground truth:
- Minimum total energy: -1
- Number of optimal conformations: 1
- Selected rotamers:
  - T0_R2
  - T1_R1
- Score calculation:
  - local energies: 1 + 0 = 1
  - pair interaction: -2
  - total energy: -1
