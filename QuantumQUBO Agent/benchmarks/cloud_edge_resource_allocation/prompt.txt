Cloud/Edge Resource Allocation Problem

Given tasks and heterogeneous compute nodes, assign each task to exactly one
node. Each assignment has a cost, and each node has a capacity.

Task:
Assign every task to a cloud or edge node while respecting node capacities.

Objective:
Minimize total assignment cost.

Goal:
Return the task-node assignment and minimum cost.

QUBO convention:
Use one binary variable per task-node pair. One-hot penalties enforce one node
per task, and capacity penalties discourage overloaded nodes.
