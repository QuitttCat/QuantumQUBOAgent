CLIQUE COVER TEST CASES

Source inspiration:
Andrew Lucas,
"Ising formulations of many NP problems",
Frontiers in Physics 2, Article 5, 2014.
DOI: https://doi.org/10.3389/fphy.2014.00005

Benchmark simplification:
These instances are kept small so the natural one-hot vertex-to-clique
encoding (x_{v,k} for v vertices, k cliques) yields at most ~12 binary
variables, allowing exact 2^n verification of the QUBO ground state.


========================================================================
TEST CASE 1: Triangle (single clique covers all)

Vertices: 0, 1, 2
Edges:
- 0, 1
- 1, 2
- 0, 2
Maximum number of cliques allowed: 2

Ground truth:
- Minimum clique cover size: 1
- One optimal cover:
  - Clique C0: 0, 1, 2


========================================================================
TEST CASE 2: Edge plus isolated vertex (two cliques needed)

Vertices: 0, 1, 2
Edges:
- 0, 1
Maximum number of cliques allowed: 2

Ground truth:
- Minimum clique cover size: 2
- One optimal cover:
  - Clique C0: 0, 1
  - Clique C1: 2
