Asset Exchange Cycle Selection Problem

Given a set of candidate asset exchanges, choose a compatible subset of
exchanges. Each exchange can involve two or more participants and has a surplus
score. Two exchanges conflict if they use the same participant or the same asset.

Task:
Choose a set of non-conflicting asset exchanges.

Objective:
Maximize total exchange surplus.

Goal:
Return the selected exchanges and maximum total surplus.

QUBO convention:
Use one binary variable per candidate exchange. Quadratic penalties prevent
selecting two exchanges that share a participant or asset. This is a small
QUBO-style simplification of the broader QUBO-Plus asset exchange framework.
