K-SAT TEST CASES

Source inspiration:
Jonas Nuesslein, Thomas Gabor, Claudia Linnhoff-Popien, and Sebastian Feld,
"Algorithmic QUBO Formulations for k-SAT and Hamiltonian Cycles",
arXiv:2204.13539, 2022.
DOI: https://doi.org/10.48550/arXiv.2204.13539

Benchmark simplification:
These cases use small CNF formulas and score assignments by unsatisfied-clause
count.


========================================================================
TEST CASE 1: Satisfiable 3-SAT formula

Variables: x0, x1, x2

Clauses:
- C0: x0 OR x1 OR x2
- C1: NOT x0 OR x1 OR x2
- C2: x0 OR NOT x1 OR x2
- C3: x0 OR x1 OR NOT x2

Ground truth:
- Minimum unsatisfied clauses: 0
- One satisfying assignment:
  - x0 = true
  - x1 = true
  - x2 = true


========================================================================
TEST CASE 2: Unsatisfiable 1-SAT core

Variables: x0

Clauses:
- C0: x0
- C1: NOT x0

Ground truth:
- Minimum unsatisfied clauses: 1
- Number of optimal assignments: 2
- Optimal assignments:
  - x0 = true
  - x0 = false
