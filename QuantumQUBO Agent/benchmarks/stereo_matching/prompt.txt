Stereo Matching Disparity Assignment Problem

Given pixels in a scanline and a discrete set of disparity labels, choose one
disparity for each pixel. Each pixel-disparity pair has a data cost. Neighboring
pixels pay a smoothness penalty when their disparities differ.

Task:
Assign exactly one disparity label to every pixel.

Objective:
Minimize:

  total stereo energy =
    sum of selected data costs
    + sum of smoothness penalties between neighboring pixels.

Goal:
Return the disparity assignment and the minimum stereo energy.
