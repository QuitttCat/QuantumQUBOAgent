STEREO MATCHING TEST CASES

Source inspiration:
William Cruz-Santos, Salvador E. Venegas-Andraca, and Marco Lanzagorta,
"A QUBO Formulation of the Stereo Matching Problem for D-Wave Quantum
Annealers",
Entropy 20(10), 786, 2018.
DOI: https://doi.org/10.3390/e20100786

Benchmark simplification:
The instances are one-dimensional scanlines. The data term is provided directly,
and each adjacent pair pays a unit smoothness penalty if the chosen disparities
differ.


========================================================================
TEST CASE 1: Three pixels, binary disparity

Pixels: P0, P1, P2
Disparities: D0, D1
Smoothness penalty for unequal neighboring disparities: 1

Data costs:
- P0: D0=0, D1=2
- P1: D0=1, D1=0
- P2: D0=2, D1=0

Neighbor pairs:
- P0, P1
- P1, P2

Ground truth:
- Minimum stereo energy: 1
- Number of optimal assignments: 1
- Optimal disparity assignments:
  - P0=D0, P1=D1, P2=D1


========================================================================
TEST CASE 2: Four pixels, binary disparity with smooth plateau

Pixels: P0, P1, P2, P3
Disparities: D0, D1
Smoothness penalty for unequal neighboring disparities: 2

Data costs:
- P0: D0=0, D1=3
- P1: D0=1, D1=0
- P2: D0=2, D1=0
- P3: D0=3, D1=0

Neighbor pairs:
- P0, P1
- P1, P2
- P2, P3

Ground truth:
- Minimum stereo energy: 2
- Number of optimal assignments: 1
- Optimal disparity assignments:
  - P0=D0, P1=D1, P2=D1, P3=D1
