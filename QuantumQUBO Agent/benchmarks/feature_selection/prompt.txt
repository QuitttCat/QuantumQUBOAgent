QUBO Feature Selection Problem

Given a set of candidate features, choose exactly K features. Each feature has
a relevance score with respect to the prediction target. Each pair of features
has a redundancy penalty.

Task:
Choose exactly K features.

Objective:
Maximize:

  total score =
    sum of selected feature relevance scores
    - sum of redundancy penalties between selected feature pairs.

Goal:
Return the selected features and the maximum total score.

QUBO convention:
Use one binary variable per feature. The exactly-K constraint is encoded with a
quadratic penalty. If using minimization, minimize the negative feature score
plus the cardinality penalty.
