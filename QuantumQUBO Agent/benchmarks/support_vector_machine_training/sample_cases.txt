SUPPORT VECTOR MACHINE TRAINING TEST CASES

Source inspiration:
Prasanna Date, Davis Arthur, and Lauren Pusey-Nazzaro,
"QUBO formulations for training machine learning models",
Scientific Reports 11, Article 10029, 2021.
DOI: https://doi.org/10.1038/s41598-021-89461-4

Benchmark simplification:
The full formulation encodes SVM weights using binary precision. These cases
provide candidate binary weight settings with precomputed regularized loss.
